

# 📚 Book-Recommender

**Ever finished a great book, craved more, but had no idea where to look next?**

The **Book-Recommender** is a lightweight Python-based CLI tool designed to solve the "reader's block." By taking your age and preferred genres into account, it delivers curated suggestions to help you find your next favorite page-turner.


##  Features
* **Age-Appropriate Filtering:** Segregates recommendations into categories (Children, YA, Adult) to ensure the content fits the reader.
* **Genre-Specific Logic:** Tailors results based on user interest—from Sci-Fi and Fantasy to History and Biographies.
* **Instant Recommendations:** A clean, fast Command Line Interface (CLI) that gives you an answer in seconds.
  

##  Built With

* **Google Books**
